# 日語文法整理（N5–N4）

> 這份整理涵蓋索引中出現的所有文法項目，按「功能」分類，方便對照學習。
> 符號說明：**V**＝動詞、**N**＝名詞、**A**＝い形容詞、**Na**＝な形容詞。

---

## 一、基本助詞

| 文法 | 意思 | 例句 |
|---|---|---|
| **～が** | 主語標記／但是（逆接） | 雨が降る（下雨了）／高いですが、買います（雖然貴，但買） |
| **～か** | 疑問；～か～か（選擇） | 行きますか？／コーヒーか紅茶か（咖啡或紅茶） |
| **N₁かN₂（か）** | N₁或N₂ | ペンか鉛筆を貸して（借我筆或鉛筆） |
| **から** | 從～；因為～ | 9時から（從9點起）／寒いから（因為冷） |
| **N₁からN₂まで** | 從N₁到N₂ | 東京から大阪まで |
| **Nで** | 在～(地點)／用～(手段) | 学校で勉強する／バスで行く |
| **Naで** | 形容動詞中止形（而且） | 静かで、きれいな部屋 |
| **～て／～で** | 動詞／い形容詞的中止連接 | 朝起きて、顔を洗う |
| **Nと** | 和～ | 友だちと話す |
| **Nと（いっしょに）** | 和～一起 | 母といっしょに行く |
| **～の** | 的；～的東西 | 私の本／赤いの（紅色的那個） |
| **N₁の～N₂** | 連接兩個名詞 | 日本語の先生 |
| **Nを** | 受詞標記 | パンを食べる |
| **Nも** | 也；竟然(數量多) | 私も行く／3時間も待った |
| **N₁もN₂も** | N₁和N₂都 | 父も母も元気だ |
| **N₁はN₂が～** | 大主語＋小主語 | 象は鼻が長い（象鼻子長） |
| **Nぐらい／Nくらい** | 大約～ | 1時間ぐらい |
| **Nだけ** | 只有～ | 一人だけ来た |
| **Nしか…ない** | 只有～(僅此，含否定) | 千円しかない（只有一千日圓） |
| **Nばかり** | 淨是～、總是～ | 遊んでばかりいる（光在玩） |
| **Nほど** | 像～那種程度 | 死ぬほど疲れた（累得要死） |

## 二、疑問詞與相關表達

| 文法 | 意思 | 例句 |
|---|---|---|
| **なぜ／なぜ～か** | 為什麼 | なぜ来なかったの？ |
| **どう／どうやって** | 怎麼樣／怎麼做 | どうやって行く？ |
| **いつ～か** | 什麼時候(不確定) | いつか日本へ行きたい（總有一天） |
| **だれか／だれが～か** | 某人／誰～呢 | だれか来た（有人來了） |
| **だれでも** | 任何人都 | だれでもできる |
| **だれも～ない** | 誰也不 | だれも来なかった |
| **なにも～ない** | 什麼也不 | 何も食べない |
| **なんでも** | 什麼都 | 何でも食べる |
| **どこが～か** | 哪裡～呢 | どこが悪いか |
| **～かどうか** | 是否～ | 来るかどうか分からない |

## 三、動詞基本變化

| 文法 | 意思 | 例句 |
|---|---|---|
| **Vます／Vています** | 禮貌形現在式／進行式 | 食べます／食べています |
| **Vたい** | 想做～ | 行きたい |
| **Vない？** | 要不要～(邀約口語) | 一緒に行かない？ |
| **～んです／～のだ** | 解釋、強調語氣 | 頭が痛いんです |

---

## 四、て形＋補助動詞

| 文法 | 意思 | 例句 |
|---|---|---|
| **Vている** | 正在／狀態持續 | 雨が降っている |
| **Vてある** | (有意)做好的狀態 | 窓が開けてある（窗戶被開著） |
| **Vておく** | 預先做好 | 予約しておく |
| **Vていく** | 漸漸～下去／去做 | 増えていく |
| **Vてくる** | 漸漸～起來／來做、去而復返 | 買ってくる（買來） |
| **Vてみる** | 試著做～ | 食べてみる |
| **Vてしまった／Vちゃった** | 做完了／不小心做了(後悔) | 全部食べてしまった／食べちゃった |
| **Vてから** | 做完～之後再 | 食べてから行く |
| **Vて（すぐに）** | 一～就立刻 | 帰ってすぐ寝た |

---

## 五、授受動詞（給予／接受）

| 文法 | 意思 | 例句 |
|---|---|---|
| **あげる／Vてあげる** | (我)給別人／替別人做 | 友だちに本をあげる |
| **くれる** | 別人給(我) | 友だちが本をくれた |
| **もらう／Vてもらう** | (我從別人)得到／請別人做 | 母に買ってもらう |
| **Vてやる** | (對下位者)替～做 | 犬に餌をやる |
| **Vてさしあげる** | 「あげる」謙讓敬語 | 先生に教えてさしあげる |
| **Vてくださる／Vていただく** | 「くれる／もらう」尊敬・謙讓 | 先生が教えてくださる |
| **Vてくださいませんか** | 能否請您～(委婉請求) | 教えてくださいませんか |
| **Vていただけませんか／Vてもらえませんか** | 能否請您～(更客氣) | 待っていただけませんか |

---

## 六、許可・禁止・義務・不必要

| 文法 | 意思 | 例句 |
|---|---|---|
| **Vてもいい／Nでもいい** | 可以～ | 帰ってもいい |
| **Vてもかまわない／Nでもかまわない** | ～也沒關係 | 行ってもかまわない |
| **Vてはいけない／Vちゃいけない** | 不可以～(禁止) | 入ってはいけない |
| **Vなくてはいけない／Vなければならない／Vないといけない** | 必須～ | 行かなければならない |
| **Vなくてもいい／Vなくてもかまわない** | 不做～也可以 | 来なくてもいい |
| **Nじゃなくてもいい／Nじゃなくてもかまわない** | 不是～也可以 | 新品じゃなくてもいい |
| **Vないで** | 不做～(而做後項) | 朝ごはんを食べないで行く |

---

## 七、假定・條件

| 文法 | 意思 | 例句 |
|---|---|---|
| **～と／Vると** | 一～就(必然結果) | 春になると花が咲く |
| **～たら** | 如果～的話／～之後 | 雨が降ったら中止 |
| **～ば** | 如果～(假設) | 安ければ買う |
| **～なら** | 如果是～的話(主題假設) | 日本へ行くなら京都がいい |
| **Vないと** | 不～的話(就不行) | 急がないと（不快點不行） |

---

## 八、比較

| 文法 | 意思 | 例句 |
|---|---|---|
| **N₁よりN₂のほうが～** | N₂比N₁更～ | 犬より猫のほうが好き |
| **～ほう** | ～(方面、那一邊) | こっちのほうがいい |
| **N₁はN₂とおなじくらい～** | N₁和N₂差不多一樣～ | 兄は父と同じくらい背が高い |
| **N₁はN₂ほど～ない** | N₁不像N₂那麼～ | 今日は昨日ほど寒くない |
| **N₁とN₂とどちら（のほうが）～** | N₁和N₂哪個比較～ | コーヒーと紅茶とどちらがいい |
| **N₁とN₂とN₃でどれが～** | N₁N₂N₃中哪個最～ | この中でどれが一番安い |

---

## 九、推量・傳聞・樣態

| 文法 | 意思 | 例句 |
|---|---|---|
| **～だろう／～でしょう** | 大概～吧 | 明日は晴れるでしょう |
| **～かもしれない** | 也許～ | 雨が降るかもしれない |
| **～はず／～はずがない（～はずない／～はずはない）** | 應該～／不可能～ | 来るはずだ／知らないはずがない |
| **～そう（樣態）** | 看起來～ | おいしそう（看起來好吃） |
| **～そうだ（傳聞）** | 聽說～ | 雨が降るそうだ |
| **～ようだ／Nのようだ／Nみたいだ** | 好像～、像～一樣 | 風邪のようだ／夢みたいだ |
| **～とおもう** | 我認為～ | 行こうと思う |
| **Vようとおもう** | 打算要～ | 来年留学しようと思う |

## 十、使役・被動・可能

| 文法 | 意思 | 例句 |
|---|---|---|
| **Vさせる** | 使役：讓／叫～做 | 子どもに掃除させる |
| **Vさせてください** | 請讓我～ | 説明させてください |
| **Vさせられる** | 使役被動：被迫～ | 食べさせられる（被迫吃） |
| **Vられる** | 被動／可能／尊敬(三義) | 怒られる(被罵)／食べられる(能吃) |
| **きける** | (耳朵)聽得到、能問 | 話が聞ける |
| **きこえる** | 聽得見(自然傳入耳) | 音楽が聞こえる |

---

## 十一、敬語（尊敬・謙讓）

| 文法 | 意思 | 例句 |
|---|---|---|
| **おVになる** | 尊敬語(對方動作) | 先生がお話しになる |
| **おVする／おVいたす／いたす** | 謙讓語(自己動作) | お持ちします／お持ちいたします |
| **なさる** | 「する」的尊敬語 | どうなさいますか |

---

## 十二、時間・順序・並行

| 文法 | 意思 | 例句 |
|---|---|---|
| **Vるとき／Vたとき** | ～的時候(做之前／做之後) | 出かけるとき／着いたとき |
| **Vるまえ（に）** | 做～之前 | 寝る前に歯を磨く |
| **Vたあと（で）** | 做～之後 | 食べたあとで散歩する |
| **V₁ながらV₂** | 一邊～一邊～ | 音楽を聞きながら勉強する |
| **～まま** | 保持～的狀態 | 電気をつけたまま寝た |
| **Nまで／Nまでに** | 到～為止／在～之前(期限) | 5時まで／5時までに |
| **まだ～／もう～** | 還～／已經～ | まだ食べていない／もう食べた |
| **ずっと** | 一直、～得多 | ずっと待っていた |

## 十三、開始・持續・結束・程度・變化

| 文法 | 意思 | 例句 |
|---|---|---|
| **Vはじめる** | 開始～ | 雨が降りはじめる |
| **Vつづける** | 持續～ | 走り続ける |
| **Vおわる** | ～完 | 食べ終わる |
| **Vだす** | 突然開始～ | 泣き出す |
| **～すぎる** | 過於～ | 食べすぎる |
| **～なる／なる** | 變得～ | 寒くなる／大人になる |
| **Vようにする** | 努力做到／設法～ | 毎日運動するようにする |
| **Vようになる** | 變得能夠／變成～ | 泳げるようになった |
| **Vにくい／Vやすい** | 難～／容易～ | 読みにくい／読みやすい |

## 十四、經驗・偶發

| 文法 | 意思 | 例句 |
|---|---|---|
| **Vたことがある／Vたことがない** | 曾經～／不曾～ | 日本へ行ったことがある |
| **Vることがある／Vないことがある** | 有時會～／有時不～ | 遅れることがある |

## 十五、目的・理由

| 文法 | 意思 | 例句 |
|---|---|---|
| **～ために／～ため（に）** | 為了～(目的)／因為～(原因) | 健康のために運動する |
| **Vにいく／Vにくる／Vにかえる** | 去／來／回去做～ | 買いに行く |
| **～ので** | 因為～(客觀禮貌) | 風邪なので休む |
| **～のに** | 明明～卻 | 勉強したのに落ちた |
| **～からです** | 是因為～ | 高いからです |

## 十六、列舉

| 文法 | 意思 | 例句 |
|---|---|---|
| **～し、（～し）** | 既～又～(並列理由) | 安いし、おいしい |
| **N₁も～し、N₂も～** | 又～又～ | 雨も降るし、風も強い |
| **Vたりする／V₁たりV₂たりする** | 又～又～(列舉動作) | 食べたり飲んだりする |
| **N₁とかN₂（とか）** | ～啦～啦(口語列舉) | りんごとかバナナとか |
| **N₁やN₂（など）** | ～和～等等 | 本やノートなど |

## 十七、引用・名詞化・意志

| 文法 | 意思 | 例句 |
|---|---|---|
| **～という／～っていう** | 叫做～、所謂的 | 田中という人 |
| **～というN** | 叫做～的(名詞) | 桜という花 |
| **～ということ** | ～這件事 | 行くということ |
| **～こと／〜さ** | 名詞化(抽象事／程度) | 泳ぐこと／高さ |
| **Vのは～** | ～的(這件事)是 | 走るのは楽しい |
| **Vところ** | 正要／正在／剛～ | 今出かけるところ |
| **Vつもり** | 打算～ | 行くつもりだ |
| **Vよう** | 意志形(～吧) | 行こう |
| **Vようにいう** | 叫(某人)做～ | 早く来るように言う |
| **Vなさい** | (命令)請～ | 早く寝なさい |

### 學習建議
- 先掌握「て形＋補助動詞」「授受動詞」「許可／禁止／義務」「條件句（と・たら・ば・なら）」這四大塊，是 N4 最常考的部分。
- 索引裡的數字是這本問題集的**題號**，可依此回去做對應練習。
- 容易混淆的成對文法（例：にくい／やすい、と／たら／ば／なら、そう樣態／そう傳聞），建議放在一起對比記。
