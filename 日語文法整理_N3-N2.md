# 日語文法整理（N3–N2 進階）

> 這本比前一本（N5–N4）程度更高，屬於 **JLPT N3（部分 N2）** 句型。
> 一樣按「功能」分類，每條附中文意思與簡短例句。索引裡的數字是這本問題集的**題號**。
> 符號：**V**＝動詞、**N**＝名詞、**A**＝い形容詞、**Na**＝な形容詞、a/b＝前項／後項。

---

## 一、推量・判斷（覺得、應該、一定）

| 文法 | 意思 | 例句 |
|---|---|---|
| **～はず** | 照理說應該～ | 彼は来るはずだ |
| **～べきだ／～べきではない** | 應該／不應該(義務、勸告) | 約束は守るべきだ |
| **～に違いない** | 一定是～(強烈推測) | 彼が犯人に違いない |
| **～に決まっている** | 肯定就是～(主觀斷定) | 嘘に決まっている |
| **～だろうと思う** | 我想大概～吧 | 雨が降るだろうと思う |
| **おそらく～だろう** | 恐怕／大概～吧 | おそらく無理だろう |
| **もしかすると～かもしれない** | 說不定～ | もしかすると遅れるかもしれない |
| **～まい／～まいか** | 大概不會～／是不是不～(否定推量) | 二度と来るまい |
| **～わけだ** | 難怪～、也就是說～ | だから怒ったわけだ |
| **（つまり）～ということだ** | 也就是說～、意思是～ | つまり中止ということだ |
| **～とのことだ／～んだって** | 聽說～(傳聞) | 来月帰国するとのことだ |

## 二、否定・部分否定

| 文法 | 意思 | 例句 |
|---|---|---|
| **必ずしも～とは限らない** | 未必一定～ | 高い物が必ずしもいいとは限らない |
| **～わけがない／～わけはない／～わけない** | 不可能～ | 知らないわけがない |
| **～わけではない** | 並不是說～(部分否定) | 嫌いなわけではない |
| **～わけにはいかない／～わけにもいかない** | (情理上)不能～ | 休むわけにはいかない |
| **～ないことはない／～ないこともない** | 也不是不～、並非不能 | 食べられないことはない |
| **Nほど…はない** | 沒有比～更…的 | 富士山ほど美しい山はない |
| **めったに～ない** | 很少～、難得～ | めったに怒らない |
| **ちっとも～ない／少しも～ない／全く～ない** | 一點也不～ | ちっとも分からない |
| **決して～ない／絶対に～ない** | 絕對不～ | 決して許さない |

## 三、假定・條件

| 文法 | 意思 | 例句 |
|---|---|---|
| **～さえ…ば** | 只要～就～ | お金さえあればいい |
| **（もし）～としたら／～とすれば** | 假如～的話 | もし宝くじが当たったとしたら |
| **（もし）～としても** | 即使～也 | 反対されたとしても行く |
| **Nにしたら／Nにすれば** | 站在～的立場來看 | 親にすれば心配だ |
| **～ば…のに／～たら…のに** | 要是～就好了(可惜現實相反) | お金があれば買えるのに |

## 四、逆接・對比・讓步

|---|---|---|
| **～くせに** | 明明～卻(帶責難語氣) | 知っているくせに教えない |
| **～わりには** | 雖然～但出乎意料 | 値段のわりにはおいしい |

## 五、時間・契機・時點

| 文法 | 意思 | 例句 |
|---|---|---|
| **～うちに** | 趁～的時候 | 熱いうちに食べて |
| **～最中に** | 正在～的時候(突發) | 食事の最中に電話が来た |
| **Vたとたん（に）** | 一～就立刻(意外) | 立ち上がったとたん倒れた |
| **Vた（か）と思うと／思ったら** | 才剛～就馬上 | 泣いたかと思うと笑った |
| **Vようとしたとき（に）** | 正要做～的時候 | 出かけようとしたとき雨が降った |
| **～ついで（に）** | 順便～ | 買い物のついでに寄った |
| **Vる／Nのたびに** | 每次～就 | 会うたびに思い出す |
| **N（っ）きり** | 自從～就一直(沒下文) | 出かけたきり帰らない |
| **Vっぱなし** | 一直保持～(放著不管) | 電気をつけっぱなし |

## 六、因果・理由

| 文法 | 意思 | 例句 |
|---|---|---|
| **～おかげで** | 多虧～(好結果) | 君のおかげで助かった |
| **～せいで／～せいか** | 都怪～(壞結果)／也許因為～ | 雨のせいで中止／年のせいか疲れる |
| **～ことから** | 由於～、從～來看 | この事故が起きたことから |
| **～ものだから／（だって）～もの／～もん** | 因為～嘛(辯解語氣) | 子どもだもの、仕方ない |
| **～ため（に）** | 因為～／為了～ | 事故のため遅れた |

| 文法 | 意思 | 例句 |
|---|---|---|
| **それで／ですから** | 因此、所以 | 雨だった。それで中止した |
| **そのため** | 因此(較正式) | 事故。そのため遅れた |
| **それでも／それなのに** | 儘管如此／明明那樣卻 | 雨だ。それでも行く |

### 學習建議
- 這一階段的關鍵在「**近義句型的細微差別**」。最容易混的幾組請對比記：
  - **わけ系**：わけがない(不可能)／わけではない(並非)／わけにはいかない(情理上不能)。
  - **に対して／について／に関して**：對象 vs 主題 vs(較正式)關於。
  - **はず／べき／に違いない／に決まっている**：推測強度與「義務」之分。
  - **せいで／おかげで**：壞結果 vs 好結果。
- 數字是這本問題集的**題號**，可回去做對應練習。
- 這份和前一本（N5–N4）合起來，基本涵蓋 N5 到 N3 的句型骨架。
